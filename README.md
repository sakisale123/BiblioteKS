# BiblioteKS

## Opis projekta

Biblioteka Management System je softverska aplikacija namenjena upravljanju bibliotekom. Projekat je trenutno u fazi razvoja i ima za cilj da omogući jednostavno upravljanje knjigama, članovima i pozajmicama.

## Funkcionalnosti (trenutno u razvoju)

* Dodavanje i brisanje knjiga
* Pregled dostupnih knjiga
* Evidencija članova biblioteke
* Pozajmljivanje i vraćanje knjiga
* Jednostavan pregled stanja biblioteke

## Tehnologije

* Java
* Objektno orijentisano programiranje (OOP)

## Status projekta

🚧 Projekat je trenutno u razvoju. Neke funkcionalnosti mogu biti nedovršene ili podložne promenama.

## Planirane funkcionalnosti

* Grafički korisnički interfejs
* Pretraga knjiga
* Evidencija istorije pozajmljivanja
* Povezivanje sa bazom podataka

## Pokretanje projekta

1. Klonirati repozitorijum
2. Otvoriti projekat u Java IDE okruženju (npr. IntelliJ IDEA ili Eclipse)
3. Pokrenuti `Main` klasu
